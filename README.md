**To compile:** 
```
$ make
```
**To run:**
```
./sortingDr
```
**While running:**

- first choose the file you want to gather integers from
```
inorder.dat 

reverse.dat 

random.dat
```

- Then choose the size of the array you want to store the numbers in 

- Finally choose an algorithm from the list provided
